# Karaoke Club Booking System

## Описание

Веб-приложение для бронирования кабинок и столов в караоке-клубе с предзаказом меню.

## Функционал

- Бронирование приватных кабинок (6 типов)
- Бронирование столов в общем зале (10 столов)
- Меню с фильтрацией по аллергенам и категориям
- Предзаказ еды/напитков
- Админ-панель для управления бронированиями
- Темная/светлая тема (куки)

## Технологии

- HTML5, CSS3, JavaScript (ES6+)
- PHP, MySQL
- AJAX (fetch), JSON
- Сессии, куки

## Установка

1. Импортируйте SQL из `database.sql` в MySQL
2. Настройте подключение в `/includes/db.php`
3. Разместите файлы на веб-сервере
4. Откройте `index.php`
